Ejercicio de JavaScript - Identificación de números positivos, negativos y cero

Instrucciones: Escribe un programa en JavaScript que solicite al usuario ingresar un número y, después de almacenarlo en una variable, determine si el número ingresado es positivo, negativo o igual a cero. A continuación, deberás mostrar un mensaje en la consola y en una ventana emergente (alert) que indique el resultado.

Pasos a seguir:

Solicita al usuario que ingrese un número utilizando la función prompt y almacénalo en una variable llamada numero.

Utiliza una estructura condicional (if, else if, y else) para determinar si el número ingresado es positivo, negativo o igual a cero.

Si el número es positivo, muestra el mensaje "El número ingresado es positivo." en la consola y en una ventana emergente utilizando la función console.log y alert.

Si el número es negativo, muestra el mensaje "El número ingresado es negativo." en la consola y en una ventana emergente.

Si el número es igual a cero, muestra el mensaje "El número ingresado es igual a cero." en la consola y en una ventana emergente.

Ejecuta el programa y prueba diferentes números para asegurarte de que las condiciones se evalúan correctamente y se muestran los mensajes correspondientes.
